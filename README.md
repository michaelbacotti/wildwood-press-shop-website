# Wildwood Press Shop — Website

Single-page static landing page for [Wildwood Press Shop](https://etsy.com/shop/WildwoodPressShop),
a print-on-demand Etsy shop by [Bacotti Inc.](https://bacotti.com/).

Live at: **https://shop.bacotti.com** (Cloudflare Pages, custom domain).

## Layout

- `index.html` — single-file landing page (no build step)
- About section, brand colors, Etsy CTA, "styles we work in" grid

## Brand

- **Shop name:** Wildwood Press Shop
- **Tagline:** A print shop in the wildwoods
- **Palette:** cream `#F5EDD8`, moss green `#5A7A52`, bark `#4A3826`, sunset `#E68A4C`
- **Type:** Recoleta (headlines) + Inter (body)

## Deploy

CF Pages auto-deploys on push to `main`. No build step.
